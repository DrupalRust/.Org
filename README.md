DrupalRust||DrupalRs, dev@inRust.Drupal.unisocial.net, parent: Web Builder AI & SuperappOS & Symbiotic.AI, description: ¨A Translation of Drupal Core in Rust. Fully compatible with PHP! Use existing Contrib modules. Write your next Drupal Module in Rust! Architecture: PHP -> C++ ->(DARPA Tractor) Rust. First translates the PHP Interpreter/runtime into C or C++! Then uses Tractor to translate that into rust. Keeps some code in C++ & uses foreign functions for any PHP features not translatable.

This project is more for syntactic beauty than performance. Rust is a very well designed & thought out language. PHP had a great run, and built the Internet, but we feel now we have better language options. See also: DrupalScala for a Scala/JVM translation! We❤️Functional Languages!¨
